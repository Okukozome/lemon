# 双擎柠檬品质分类检测系统 V2.0

## 一、 项目概述

本项目是一个基于深度学习与大语言模型（LLM）的多模态检测系统，旨在解决柠檬品质的自动化检测与智能化评价问题。
系统在原有的卷积神经网络（CNN）图像分类基础上，引入了具有视觉识别能力的大模型（GLM-4.5V），形成了“本地 CNN 精准分类 + 云端 LLM 深度评价”的双擎架构。此外，系统完整封装了基于 Tkinter 的现代化多页面桌面图形用户界面（GUI），集成了外设摄像头调用、本地图片检测、TTS 语音自动播报以及检测日志 Excel 导出等丰富的实用功能。

系统能够高精度识别并区分三种类别：**优质柠檬**、**劣质柠檬**以及**无柠檬**。

---

## 二、 数据集说明

* **数据来源**: 数据集来源于 Kaggle 平台的 Lemon Quality Dataset。
* **数据规模**: 数据集总共包含 2,533 张图像，所有原始图像的分辨率均为 300 x 300 像素。
* **图像内容**: 图像主要记录了放置在混凝土表面上的柠檬。除不同品质的柠檬样本外，数据集同时包含了仅有该表面的空背景图像。
* **环境特征**: 所有的柠檬图像均在自然日光下拍摄。数据样本覆盖了略微不同的光照条件以及不同的柠檬尺寸特征。

## 三、 核心模块与技术架构

本项目采用了 `TensorFlow + OpenCV + GLM-4.5V + Tkinter` 的综合技术栈。系统按功能被模块化拆分为以下几个核心部分：

### 1. 数据增强与预处理 (`preprocess.py` & `prepare_dataset.py`)
系统在训练前使用 OpenCV 库对原始数据进行了统一的离线增强处理，以提升模型在不同光照和噪点条件下的鲁棒性。
* **色彩空间转换与 CLAHE 增强**: 将 BGR 图像转换为 LAB 色彩空间，针对决定图像亮度的 L 通道（亮度通道）应用限制对比度自适应直方图均衡化（CLAHE），参数设定为 `clipLimit=2.0`, `tileGridSize=(8, 8)`，增强图像暗部与亮部细节，完成后再转回 BGR 空间。
* **平滑去噪**: 对增强对比度后的图像，使用尺寸为 `3x3` 的高斯模糊（GaussianBlur）进行去噪处理，消除异常的高频噪点。
* **统一重采样与生成**: `prepare_dataset.py` 脚本会将所有原始图像统一缩放至 `128 x 128` 尺寸，应用上述增强算法后，批量输出至 `./lemon_dataset_enhanced` 目录供后续训练使用。
* **在线推理预处理**: GUI 系统在检测实时图像时，会调用 `preprocess_for_inference` 接口，进行完全一致的缩放与增强操作，并将数据最终转化为 `float32` 类型的 RGB 矩阵。

### 2. CNN 模型结构与训练策略 (`train.py`)
模型采用 TensorFlow/Keras 构建的 Sequential 顺序卷积神经网络，作为“第一擎”的高速精准分类。
* **数据加载与性能优化**: 使用 `image_dataset_from_directory` 加载增强后的数据集，按 80% 训练集、20% 验证集划分。结合 `tf.data.AUTOTUNE` 开启数据集的 `cache()` 缓存与 `prefetch()` 预取操作，消除 I/O 瓶颈。
* **网络架构**:
  * **输入层**: 接收 `(128, 128, 3)` 尺寸输入，首层采用 `Rescaling(1./255)` 将像素值归一化，加速模型收敛。
  * **第一层卷积池化**: 32 个 `3x3` 卷积核（ReLU激活）提取边缘、颜色等低级特征，接 `2x2` MaxPooling 降维。
  * **第二层卷积池化**: 64 个 `3x3` 卷积核提取更复杂的局部纹理特征，接 `2x2` MaxPooling。
  * **第三层卷积池化**: 128 个 `3x3` 卷积核提取高级语义特征，接 `2x2` MaxPooling。
  * **全连接与防过拟合**: 数据经过 `Flatten` 展平后，送入 128 个神经元的 `Dense` 层。为了防止过拟合，引入了 `Dropout(0.5)` 层，在训练时随机丢弃 50% 的神经元。
  * **输出层**: `Dense` 层包含 3 个神经元，采用 `Softmax` 激活函数输出各类别概率。
* **编译与训练**: 采用 `Adam` 优化器，损失函数设为 `sparse_categorical_crossentropy`（稀疏类别交叉熵），共进行 15 个 Epoch 的训练。训练后的权重保存至 `./models/lemon_cnn_model.h5`。

### 3. 多模态 AI 评价机制（双擎中的“第二擎”）
系统通过调用云端大语言模型（GLM-4.5V）为 CNN 的分类结果提供深度、趣味性及专业性的文本补充说明。
* **环境配置 (`config.py`)**: 通过 `dotenv` 加载环境变量 `.env`，读取 `API_URL` 与 `API_KEY`，保障云端调用的安全性。
* **异步 API 调用 (`pages.py` -> `call_llm_async`)**: 
  * 为防止网络请求阻塞 GUI 主线程造成界面卡顿，系统开启了独立的后台线程 (`threading.Thread`) 调用 LLM。
  * 将待检测图像通过 `utils.py` 中的 `image_to_base64` 转化为 Base64 字符串，随同 CNN 的检测结果（分类与置信度）组合成复杂的 Prompt 提示词。
  * Prompt 设定了严格的输出要求（结合图片画面、不超过50字、给出处理建议、不含 markdown 标记），并在 API 响应后通过 `after` 方法安全地回调更新 UI。

### 4. GUI 交互与外设调度 (`app.py`, `pages.py`, `utils.py`)
系统摒弃了单一窗口，采用了基于 Tkinter 面向对象的**多页面架构**，并加入了丰富的硬件交互功能。
* **框架设计 (`app.py`)**: 
  * 主类 `LemonQualityApp` 继承自 `tk.Tk`，锁定窗口大小为 `900x650`，并在底层实现了页面管理器 (`show_frame`) 和背景图渲染。
  * 集成了基于 Pandas 的检测日志管理模块。用户点击“导出日志”可一键将所有的历史检测记录（包含时间、图片来源、CNN分类、置信度、AI评价）导出为 `.xlsx` Excel 文件。
* **多页面调度 (`pages.py`)**:
  * **`BasePage`**: 提供底层 Canvas 画布，承载全屏背景图（透视效果）。
  * **`HelpPage`**: 系统帮助页，提供详细的悬浮卡片式系统操作说明。
  * **`DetectionPage`**: 核心工作区，分为左侧的图像预览卡片与右侧的控制面板区。
* **摄像头智能调度 (`utils.py` & `pages.py`)**:
  * **摄像头可用性校验**: `check_camera()` 算法不仅尝试打开摄像头设备，还通过读取连续帧并计算像素标准差（`std_dev`）和水平梯度均值（`mean_diff`）来判断摄像头是否被其他程序占用或画面全黑/卡死。
  * **动态弹窗抓拍**: 用户点击“打开摄像头”后，系统弹出一个独立窗口实时播放摄像机画面（30ms 刷新率），主界面的按钮动态切换为红色的“拍照”状态。完成抓拍后自动保存到 `./test_images/temp_capture.jpg` 并释放相机资源。
* **异步语音播报 (`utils.py`)**: 
  * 依托 `pyttsx3` 库，当 LLM 返回评价结果且无异常时，系统会启动后台守护线程 (`daemon=True`) 自动进行中文语音播报，实现流畅的交互体验。

## 四、 系统部署与运行流程

1. **环境依赖准备**: 
   * 确保安装相关依赖：`pip install -r requirements.txt`。
   * 在项目根目录创建 `.env` 文件，并填入如下格式的云端大模型的 API 配置：
     ```ini
     # 本项目使用白山智算平台提供的中转 API 服务
     API_URL=https://api.edgefn.net/v1/chat/completions
     API_KEY=YOUR_API_KEY
     ```
2. **数据准备与增强**: 
   * 将原始分类图像按文件夹名（`bad_quality`, `empty_background`, `good_quality`）放置在 `./lemon_dataset` 目录下。
   * 运行 `python prepare_dataset.py`，将生成 `128x128` 的增强图像并存入 `./lemon_dataset_enhanced` 目录。
3. **模型训练**: 
   * 运行 `python train.py` 执行模型训练。训练完成后，模型权重文件 `.h5` 将自动生成在 `./models/` 目录下。
4. **启动双擎检测系统**: 
   * 在终端执行入口脚本：**`python main.py`**
   * **操作流程**:
     1. 点击【本地图片】导入或点击【打开摄像头】进行实时拍照。
     2. 点击【开始双擎检测】。系统将秒级输出 CNN 分类与置信度。
     3. 随后系统自动请求云端 GLM-4.5V，数秒内返回深度 AI 评价，并自动进行语音播报。
     4. 检测完毕后，可点击右上角【导出日志】将历史记录归档至 Excel。
